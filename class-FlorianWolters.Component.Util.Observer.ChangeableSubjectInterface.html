Interface ChangeableSubjectInterface

The interface FlorianWolters\Component\Util\Observer\ChangeableSubjectInterface can be implemented by a class to represent an observable object, or "data" in the model-view paradigm.

It can be implemented to represent an object that the application wants to have observed.

The interface FlorianWolters\Component\Util\Observer\ChangeableSubjectInterface implements the Subject part of the Observer behavioural design pattern.
