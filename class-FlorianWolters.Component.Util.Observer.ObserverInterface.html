Interface ObserverInterface

The interface FlorianWolters\Component\Util\Observer\ObserverInterface can be implemented by a class when it wants to be informed of changes in observable objects.

The interface FlorianWolters\Component\Util\Observer\ObserverInterface implements the Observer part of the Observer behavioural design pattern.
